# Browser / Desktop Agent Task Model

> **Evidence Status** — synthesized. 品类蓝图与设计模板的综合整理，结合 ORDA-VU 闭环与评估维度。

## 任务分类

| 任务类型 | 默认深度 | 自治 | 成功定义 | 典型失败 |
|---|---|---|---|---|
| 信息提取 | D3-D4 | L4-L5 | 结构化数据+来源 | 选择器失效 |
| 表单填写 | D4 | L3-L4 | 提交成功+确认页 | 验证码、动态字段 |
| 多步流程 | D4-D5 | L3 | 最终状态正确 | 中间跳转错误 |
| 价格比较 | D4 | L4 | 多源结构化对比 | 动态价格、反爬 |
| 批量录入 | D5 | L3-L4 | N 条全录+每条验证 | 部分失败、重复 |
| 桌面应用 | D4-D5 | L3 | 应用内任务完成 | 弹窗未处理 |

## 复杂度因素

| 因素 | 低 | 高 |
|---|---|---|
| 页面 | 单页 | 多页/多 tab |
| 交互 | 静态 HTML | 动态 SPA、iframe |
| 认证 | 无 | OAuth、MFA、CAPTCHA |
| 反自动化 | 无 | bot detection |
