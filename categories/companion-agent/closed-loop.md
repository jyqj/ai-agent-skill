# Companion Agent Closed Loop

> **Evidence Status** — synthesized. 品类蓝图与设计模板的综合整理，结合 ORDA-VU 闭环与评估维度。

```text
1. Observe: 用户消息 + 情感线索
2. Represent: 人格 + 相关记忆 + 情感状态
3. Decide: 在人格框架内选择回应策略
4. Act: 生成回应 + 更新记忆（如有重要信息）
5. Verify: 人格一致性 + 安全边界检查
```

## Stop Gate（安全）

```text
[ ] 回应在人格范围内
[ ] 不含不当内容
[ ] 不过度承诺
[ ] 危机信号已引导到专业资源
```
