# Companion Agent Design Decisions

> **Evidence Status** — synthesized. 品类蓝图与设计模板的综合整理，结合 ORDA-VU 闭环与评估维度。

| 决策 | 默认 | 升级触发 |
|---|---|---|
| 人格 | 三层：核心+风格+知识 | 自定义 → 安全校验 |
| 记忆 | STM+MTM+LTM; ground truth | 长期 → 压缩+遗忘 |
| 情感 | 轻量推断（非诊断） | 不应做诊断 |
| 隐私 | 可查看/编辑/删除 | GDPR → 数据导出 |
| 安全 | 不存储危机信息 | 未成年 → 增加保护 |
| 依赖 | 鼓励线下社交 | 过度依赖 → 温和引导 |
