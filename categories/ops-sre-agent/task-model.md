# Ops / SRE Agent Task Model

> **Evidence Status** — synthesized. 基于品类架构模板和 Ops/SRE 场景的通用设计。

## 任务分类

| 任务类型 | 深度 | 自治 | 成功定义 | 典型失败 |
|---|---|---|---|---|
| 告警分诊 | D3-D4 | L4-L5 | 分类准确+噪音过滤 | 误分类 |
| 根因分析 | D4-D5 | L4 | 假设+证据+置信度 | 误导方向 |
| Runbook 执行 | D4-D5 | L3-L4 | 按序执行+验证 | 环境差异 |
| 自动缓解 | D5 | L3 | 执行+验证+回滚准备 | 引入新问题 |
| 发布评估 | D4-D5 | L3-L4 | canary 分析+建议 | 统计误判 |
| 回滚 | D5 | L3 | 执行+恢复验证 | 数据不一致 |
| 事后复盘 | D4 | L4-L5 | timeline+根因+action items | 归因错误 |

## 时间压力

| 紧急度 | 时间窗口 | 决策模式 |
|---|---|---|
| P0 | 分钟级 | 快速缓解 → 后续分析 |
| P1 | 小时级 | 分析+缓解并行 |
| P2 | 天级 | 深度分析 → 规划修复 |
