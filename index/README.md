# 交叉索引

> **Evidence Status** — synthesized. 跨项目观察归纳。

本目录下的文件用于快速回答五类问题：

1. 这套知识树的主干怎么走？
2. 某个 Agent 设计问题应该看哪个范式？
3. 哪类 Agent 需要哪些模块？
4. 某个能力域应该看哪些 plane / project？
5. 设计评审时有哪些必问问题？

当前重点索引：

- `mainline-map.md`：概念 → 范式 → 品类 → 模块 → 项目的主干地图
- `paradigm-map.md`：设计问题到范式文档的映射
- `architecture-map.md`：运行时架构入口
- `capability-map.md`：能力域到模块/项目的映射
- `category-module-map.md`：品类到必备模块的映射
- `category-pattern-matrix.md`：品类到设计模式的映射
- `project-module-map.md`：参考项目到模块的映射
- `design-checklist.md`：设计评审 checklist
