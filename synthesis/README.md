# 综合分析

> **Evidence Status** — synthesized. 本页来自项目横向分析与本 skill 的架构抽象汇总，用于跨品类比较和设计取舍。

> **定位**：跨项目的横向比较、趋势判断和开放问题。不止于单个项目的视角。

这一层回答的问题包括：

- 不同品类的 Agent 在同一设计维度上做了什么不同选择？
- 哪些边界划分在多个项目中反复出现？
- 主流框架的抽象与本知识库的抽象如何映射？
- 哪些问题至今没有标准答案？
